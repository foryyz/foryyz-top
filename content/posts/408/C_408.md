# C - for 408



### `***1 运算符优先级

**! > 算术运算符 > 关系运算符 > && > || > 赋值运算符**
(!, &&, || 都是逻辑运算符)

### `2 短路写法

```c
int i=0;
i && printf("You Can't See ME !\n"); // 不会执行&&后边的

int j=1;
j || printf("You Can't See ME !\n"); // 这也是短路运算
```

### `3 左值右值

```c
a+25 = b; // 这会导致编译错误
```
