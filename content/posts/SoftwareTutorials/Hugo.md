# HUGO

## 基础命令

```bash
# 查看版本
hugo version

# 初始化站点 - 在当前目录下创建新的Hugo站点
hugo new site <站点名>

# 创建新文章
hugo new <路径(posts/??)>/<文件名>.md

#编译生成静态文件并启动web服务
hugo server

# 编译生成静态文件 - 将编译所有文件并输出到public目录  
hugo

#版本和环境详细信息
hugo env
```
